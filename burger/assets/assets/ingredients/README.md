# 商品ドット絵アセット

ここに**透過PNGのドット絵**を置くと、ゲームの組み立て表示がその画像になります。
画像が無い商品は従来の手続き描画にフォールバックします。

## ファイル名（このまま使用）

| ファイル | 用途 | 目安サイズ |
| --- | --- | --- |
| `full_burger.png` | 通常のバーガー全般（正面・完成イメージ） | 512×512 程度の正方形・透過 |
| `salad.png` | グリーンサラダ | 同上 |
| `fries.png` | フライドポテト | 同上 |
| `nuggets.png` | チキンナゲット | 同上 |
| `drink.png` | ドリンク | 同上 |

## ポイント

- **正面から見た完成形**の1枚絵でOK（先日もらったような美味しそうなバーガー）。
- 背景は**透過**にしてください。
- 組み立て中は、画像を**下から徐々に出して「積み上がっていく」演出**で表示します
  （完成で全体表示）。
- まずは `full_burger.png` 1枚だけでも効果大です。
