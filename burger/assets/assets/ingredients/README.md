# 商品ドット絵アセット

ここには商品イメージ用の**透過PNGのドット絵**を置いています。
ゲームの組み立て中はこれらの完成画像を分割せず、選んだ素材を一段ずつ描画します。

## ファイル名（このまま使用）

| ファイル | 用途 | 目安サイズ |
| --- | --- | --- |
| `full_burger.png` | 通常のバーガー全般（正面・完成イメージ） | 512×512 程度の正方形・透過 |
| `salad.png` | グリーンサラダ | 同上 |
| `fries.png` | フライドポテト | 同上 |
| `nuggets.png` | チキンナゲット | 同上 |
| `drink.png` | ドリンク | 同上 |

## ポイント

- **正面から見た完成形**の1枚絵を格納します。
- 背景は**透過**にしてください。
- 組み立て中の表示には、注文データの素材レイヤーを使用します。
